# 🎬 Get Started in 30 Seconds!

## ⚡ Quick Start Commands

```bash
# Step 1: Navigate to project
cd c:\Users\anany\daansetu\frontend

# Step 2: Install dependencies (if not already done)
npm install

# Step 3: Start the development server
npm run dev

# Step 4: Open in browser
# Go to: http://localhost:5173
```

## ✨ What You'll See

When you run the app, you'll have access to:

### 🏠 Landing Page (`/`)
- Beautiful hero section
- Feature showcase
- Call-to-action buttons
- Responsive design

### 🔐 Authentication Pages
- **Login** (`/login`) - Sign in
- **Register** (`/register`) - Create account with role selection

### 📊 Donor Features
- **Dashboard** (`/donor-dashboard`) - Main donor interface
- **Add Donation** (`/add-donation`) - Post new items
- **View Donations** (`/view-donations`) - See all your donations

### 🏢 Organization Features
- **Dashboard** (`/org-dashboard`) - Browse and request donations

### 👨‍⚖️ Admin Features
- **Dashboard** (`/admin-dashboard`) - Manage platform
