## 💡 Customization Guide

### Change Colors
Edit `src/index.css` for global color scheme changes

### Add New Page
1. Create component in `src/pages/NewPage.jsx`
2. Create styles in `src/pages/NewPage.css`
3. Import in `App.jsx`
4. Add route to Routes

### Modify Styling
- Component styles: Edit corresponding `.css` file
- Global styles: Edit `src/index.css` or `src/App.css`
- Responsive: Update media queries at bottom of CSS

### Update Navigation
- Edit Navbar links in `src/components/Navbar.jsx`
- Add/modify routes in `src/App.jsx`
